# Project Context: M's Simply Sourdough Modernization

## 📋 Overview
- **Business**: Artisanal Sourdough Bakery in Cedar City, UT.
- **Brand Identity**: Premium, Sleek, Dark Artisan. 2026 Modernist aesthetic.
- **Primary Colors**: Dark (#020205), Gold (#f0c878), Purple/Gold Ambient Glows.

## 🛠️ Technical Stack
- **Architecture**: Single-page HTML/CSS/JS.
- **Layout**: Custom Bento Grid (Desktop) / Vertical Stack (Mobile).
- **Design Patterns**: Glassmorphism, Film Grain, Artisanal Tilt (Accents), Intersection Observer (Scroll Reveals).
- **Hosting**: GitHub Pages (pan-dit.github.io/msimplysourdough).
